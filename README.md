# Projeto Final APC (Algoritmos e Programação de Computadores) 2025.1 - 1º periodo

O presente projeto é um sistema desevolvido em Python utilizando o Framework Flask como projeto final da disciplina de Algoritmos e Programação de Computadores, lecionada pelo professor Alexandre Barbosa. 

#### Tecnologias Utilizadas:
* Front-End
1. HTML, CSS
2. JAVASCRIPT
3. FRAMEWORK

* Back-End
1. Python
2. Flask
3. JSON

#### Requisitos
Por motivos educacionais, não utilizamos um SGBD para incluir o conteúdo de arquivos. Sendo assim é necessário criar três arquivos .json que serão as bases de dados utilizados pelo sistema `livros.json` e 'registros.json' em `app_biblioteca/`

```
import json
with open('app_biblioteca/registros.json','r') as arquivo
    json.dump([], arquivo)
```

